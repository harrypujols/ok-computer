Welcome To Gulp Boilerplate

A not-so opinionated static site generator.

Stuff installed:

  • Gulp.js for automation and compiling.
  • Modernizr for keeping browsers in check.
  • Normalize to get your styles going without a heavy framework.

Pre-compiled languages

  • Swig as the official templating language.
  • Sass as the official css pre-processing language.
  • Coffeescript for when you need a break from JavaScript.

Stuff that has to be installed

  • Compass for sprite building, and other fun stuff with Sass.

Software used

  • Brackets the official text editor for this repo.
  • Node.js because you can't run Gulp.js without it.

How to use and configure

to run it

npm install

npm gulp

Use files in the dev folder.

to deploy

  • Create a repo subtree, using git subtree push --prefix build origin gh-pages
  • Configure the gulp-gh-pages plugin in the gulpfile to point to the repo.
  • Use the task gulp deploy. It will send the contents of the build folder to Githbu pages.
  • The URL of the project will be at username.github.io/repo.
